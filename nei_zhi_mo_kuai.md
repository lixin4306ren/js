# 文件操作内置模块

## File System（文件系统）
http://nodejs.org/api/fs.html
```
fs.readFile             // 异步操作
fs.readFileSync         // 同步操作
fs.writeFileSync
fs.createReadStream     // 创建读文件流
fs.createWriteStream    // 创建写文件流
```

## zlib (压缩)
```
var zlib = require('zlib');
zlib.createGzip()   //压缩
zlib.createGunzip() //解压
```
